# WhatsApp UI Demo

This repository contains a **lightweight WhatsApp-style UI demo** built for rapid prototyping.

## Purpose
- UI-first demo (no backend, no database)
- Mobile & tablet friendly
- Zero-cost testing
- Designed to be connected later with n8n / webhook-based systems

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript

## Status
⚠️ Demo only. No authentication, no real messaging.

## Next Steps
- Connect webhook (n8n Data Table or similar)
- Add message states (sent / delivered)
- Improve responsive behavior

---

Built for fast iteration, not perfection.
